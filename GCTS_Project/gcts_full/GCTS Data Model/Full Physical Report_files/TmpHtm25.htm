Fakt table will base on feed:

 

select * from DB_CONS_STG_DEV.STG_GCTS.V_GCTS_RESPONSE r

join DB_CONS_STG_DEV.STG_GCTS.V_GCTS_COUNTRY_CATEGORY cc on r.COUNTRYCATEGORYID=cc.COUNTRYCATEGORYID

where r._ETL_ACTIVE_FLAG=true and cc._ETL_ACTIVE_FLAG=true

---------------------------------------------------------------------------

Unique row will be defined by:
ID_+ variableid + crountrycodeid + responsename - on source side
ID + DIM_GCTS_QUESTION_MAP_KEY + DIM_GCTS_OPTIONS_KEY - on fact side

---------------------------------------------------------------------------
Columns mapping:

DIM_GCTS_QUESTION_MAP_KEY <- MD5(variableid||COUNTRYCATEGORYID )
DIM_GCTS_COUNTRY_KEY <- country - column available after joining with country category

DIM_GCTS_OPTIONS_KEY <- MD5(variableid||COUNTRYCATEGORYID||RESPONSENAME)

DIM_DATE_KEY <- calculated, to integer 01mmyyyy, based on month and year columns

other 1:1
